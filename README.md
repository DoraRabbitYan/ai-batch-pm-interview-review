# 📊 AI 产品经理多场面试横评

> 攒够多场单次复盘后，一键生成能力趋势横评和面试题库。

## 它能做什么

- 📥 自动从 VIP Docs 读取多场单次复盘文档
- 📈 四象限能力横评——标注趋势（↑进步 / ↓退步 / →持平），识别共性优缺点
- 🔥 高频失分 TOP5 排序，按频率 × 严重程度，附改进方向
- 🎯 改进优先级 Top 3，按 ROI（影响面 / 修复难度）排列
- 📄 产出两份文档：横评总结 + 题库与参考回答

## 安装

### 方式一：vskills 安装

```bash
npx vskills add DoraRabbitYan/ai-batch-pm-interview-review -y --copy
```

### 方式二：Git 克隆

```bash
git clone https://github.com/DoraRabbitYan/ai-batch-pm-interview-review.git
```

## 快速开始

前提：已通过 `ai-pm-interview-review` 攒了多场单次复盘文档。

```
帮我来个面试横评
```

自动读取复盘 → 生成横评总结（文档A）→ 可选生成题库（文档B）。

## 与单场复盘的上下游关系

```
单场问答 → ai-pm-interview-review → 单场复盘文档
                                        │
                            多场攒够 → 本工具 → 横评总结 + 题库
```

| | 单场复盘 | 多场横评 |
|------|----------|----------|
| 输入 | 原始面试对话 | 已有的复盘文档 |
| 产出 | 逐题评估 + 改进示范 | 能力趋势 + 高频失分 + 题库 |

## 输出文档

| 文档 | 内容 |
|------|------|
| 文档A：横评总结 | 场次概览、四象限能力横评、高频失分 TOP5、趋势表、改进优先级 |
| 文档B：题库与参考回答 | 全量问题按象限分类 + 改进版回答 |
